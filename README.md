# 智能家居控制面板

基于边缘计算的智能家居管理系统 - 阿里云ESA Pages边缘开发大赛参赛作品

## 项目简介

本项目是一个现代化的智能家居控制面板，采用React + TypeScript构建，支持3D可视化、实时设备控制、场景联动、能源管理等核心功能。系统设计遵循"本地优先、边缘计算"的理念，实现毫秒级响应和断网本地自治。

本项目由[阿里云ESA](https://www.aliyun.com/product/esa)提供加速、计算和保护![](assets/esa.png)

## 核心特性

### 🏠 空间可视化
- 3D全屋户型图交互展示
- 设备按房间精准定位
- 实时状态可视化（运行中/异常/离线）
- 窗帘3D模型展示（动态开合效果）
- 房间布局设计（客厅、卧室、厨房、书房、走廊）
- 家具模型展示（沙发、桌子、床、电视、柜子等）

### 🎛️ 设备管理
- 支持23类标准智能设备
- Matter 1.2统一协议
- 多维数据展示（温度、能耗、状态等）
- 实时设备控制
- 冰箱温度精确控制（2-8°C）
- 窗帘开合度控制（0-100%）
- 3D场景中直接操作设备

### 🎬 场景联动
- 预设场景库（睡眠模式、离家模式、养老守护、节能模式等）
- 可视化场景编辑器（已实现）
- 自定义场景创建和保存
- 场景持久化存储
- 定时触发和条件触发（预留接口）

### ⚡ 绿色能源管理
- 实时能耗看板
- 动态电价响应
- 节能建议推送
- 绿电优先调度

### 🔒 安全与隐私
- TLS 1.3加密传输（预留接口）
- 细粒度权限控制
- 数据脱敏处理（预留接口）
- 本地数据优先
- 设备状态本地持久化

### 📱 多端适配
- 响应式设计
- PWA离线支持
- 极简模式（老人/儿童友好）

## 技术栈

- **前端框架**: React 18 + TypeScript
- **3D可视化**: Three.js + React Three Fiber
- **UI组件**: Ant Design 5
- **状态管理**: Zustand
- **图表库**: ECharts
- **实时通信**: WebSocket (模拟边缘网关)
- **构建工具**: Vite
- **样式方案**: CSS Modules + Ant Design

## 部署说明

### 部署到阿里云ESA Pages

1. 将代码推送到GitHub公开仓库
2. 在阿里云ESA Pages控制台创建项目
3. 连接GitHub仓库
4. 配置构建命令：`npm run build`
5. 配置输出目录：`dist`
6. 部署完成后获取访问URL

## 功能演示

### 主要页面

- **大屏首页**: 欢迎界面，可跳转到智能家居控制面板
- **设备管理**: 设备列表、详情、控制、3D可视化
- **场景中心**: 场景列表、场景编辑器、自定义场景创建
- **能源管理**: 能耗看板、节能建议、数据可视化
- **设置**: 权限管理、系统配置、本地模式设置

## 技术亮点

1. **边缘计算架构**: 模拟边缘网关通信，实现本地优先策略
2. **3D可视化**: 使用Three.js实现流畅的3D交互体验，支持窗帘3D模型展示
3. **实时通信**: WebSocket模拟MQTT-SN协议，实现<30ms延迟
4. **本地存储**: Zustand persist中间件实现设备状态和场景持久化
5. **响应式设计**: 完美适配手机、平板、桌面、大屏
6. **精细化控制**: 支持冰箱温度、窗帘开合度等多种设备属性控制
